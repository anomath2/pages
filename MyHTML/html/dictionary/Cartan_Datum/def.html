Definition of Cartan Datum

PDF

Definition

A Cartan datum is a finite set $I$ equipped with a symmetric bilinear form \[ \cdot : \mathbb{Z}[I] \times \mathbb{Z}[I] \rightarrow \mathbb{Z} \]satisfying the following for all $i \in I$.
  • $i \cdot i \in 2\mathbb{Z}_{> 0}$.
  • $a_{i,j} := \frac{2 i \cdot j}{i \cdot i} \in \mathbb{Z}_{\leq 0}$ if $i \neq j$.

Associated Generalized Cartan Matrix

The matrix $(a_{i,j})_{i,j \in I}$ is a symmetrizable generalized Cartan matrix with a symmetrizing matrix $\operatorname{diag}(\frac{i \cdot i}{2})_{i \in I}$. It is called the generalized Cartan matrix of $I$.
Conversely, given a symmetrizable generalized Cartan matrix $A = (a_{i,j})_{i,j \in I}$ with a symmetrizing matrix $D = \operatorname{diag}(d_i)_{i \in I}$, one can make $I$ into a Cartan matrix by setting \[ i \cdot j := d_i a_{i,j} \text{ for all } i,j \in I. \]If one chooses another symmetrizing matrix $D' = \operatorname{diag}(d'_i)_{i \in I}$, then the resulting Cartan datum $I' = (I, \circ)$ is related to $I$ as follows: \[ d'_i i \cdot j = d_i i \circ j \text{ for all } i,j \in I. \]In particular, they are proportional.
In this way, we obtain a bijection \[ \{ \text{Cartan datum} \}/\text{proportion} \rightarrow \{ \text{symmetrizable generalized Cartan matrix} \}. \]

Memo

  • The definition above can be found in [Lus10, Section 1.1.1].
  • A Cartan datum is used to construct a quantum group.
