Definitions Concerning Cartan Datum

Definition of Cartan Data

A Cartan datum is a finite set $I$ equipped with a symmetric bilinear form \[ \cdot : \mathbb{Z}[I] \times \mathbb{Z}[I] \rightarrow \mathbb{Z} \]satisfying the following for all $i \in I$.
  • $i \cdot i \in 2\mathbb{Z}_{> 0}$.
  • $a_{i,j} := \frac{2 i \cdot j}{i \cdot i} \in \mathbb{Z}_{\leq 0}$ if $i \neq j$.
([Lus10, Section 1.1.1])

Definition of Morphisms between Cartan Data

Let
  • $I,J$: two Cartan data.
A morphism from $I$ to $J$ is a map \[ f: I \rightarrow J \]such that \[ f(i) \cdot f(j) = i \cdot j \text{ for all } i,j \in I. \]
A morphism is said to be an isomorphism if the underlying map is bijective.
A morphism $f : I \rightarrow I$ is said to be an automorphism if it is an isomorphism.

Definition of Irreducible Cartan Data

The Cartan datum $I$ is said to be irreducible if for each $i \neq j \in I$, there exists a sequence $i = i_1, i_2, \dots, i_r = j \in I$ such that $i_k \cdot i_{k+1} < 0$ for all $k =1,\dots,r-1$ ([Lus10, Section 2.1.3]).

Definition of Cartan Data of Finite Type

The Cartan datum $I$ is said to be of finite type if its Cartan matrix $(a_{i,j})_{i,j \in I}$ is positive definite ([Lus10, Section 2.1.3]).
The irreducible Cartan data of finite type are completely classified (see here).

Definition of Subdata

Let
  • $I$: a Cartan datum.
  • $J \subset I$: a subset.
Then, $J$ is equipped with a Cartan datum structure by restricting the symmetric bilinear form $\cdot$ on $I$ to $J$. The Cartan datum $J$ is called a subdatum of $I$.
The inclusion map $J \rightarrow I$ gives rise to a morphism of Cartan data.

Definition of Proportionality of Cartan Data

Let
  • $(I, \cdot)$, $(I, \circ)$: two Cartan data with the same underlying set $I$.
We say that $(I, \cdot)$ and $(I, \circ)$ are proportional if there exist $a,b \in \mathbb{Z}_{> 0}$ such that \[ a i \circ j = b i \cdot j \text{ for all } i,j \in I. \]([Lus10, Section 1.1.1])

Definition of Coxeter Graphs of Cartan Data

The Coxeter graph of the Cartan datum $I$ is the colored graph whose vertex set is $I$, and whose edges sets are defined as follows. Let $i,j \in I$. Then, there exists exactly one edge between $i$ and $j$ if and only if $a_{i,j} < 0$; in this case, the edge is labeled by $\{ |a_{i,j}|, |a_{j,i}| \}$. In addition, if $|a_{i,j}| < |a_{j,i}|$, then the edge is oriented from $i$ to $j$.
The Cartan datum is recovered from its Coxeter graph up to proportion.

Memo

  • A Cartan datum is used to construct a quantum group.
