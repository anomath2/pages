Weyl Group of a Cartan Datum

Definition

Let
  • $I$: a Cartan datum.
  • $a_{i,j} := \frac{2 i \cdot j}{i \cdot i}$ for each $i,j \in I$.
For $i \neq j \in I$, set \[ m_{i,j} := \begin{cases} 2 & \text{ if } a_{i,j} a_{j,i} = 0, \\ 3 & \text{ if } a_{i,j} a_{j,i} = 1, \\ 4 & \text{ if } a_{i,j} a_{j,i} = 2, \\ 6 & \text{ if } a_{i,j} a_{j,i} = 3, \\ \infty & \text{ if } a_{i,j} a_{j,i} \geq 4. \end{cases} \]
The braid group is the group $\mathrm{Br}$ with generators \[ \{ s_i \mid i \in I \} \]subject to the following relations for all $i \neq j \in I$ with $m_{i,j} < \infty$: \[ \underbrace{s_i s_j s_i \cdots}_{\text{$m_{i,j}$ factors}} = \underbrace{s_j s_i s_j \cdots}_{\text{$m_{i,j}$ factors}} \]
The Weyl group is the quotient group $W$ of $\mathrm{Br}$ factored by the normal subgroup generated by \[ \{ s_i^2 \mid i \in I \} \]([Lus10, Section 2.1.1]).

Properties

  • if $m_{i,j} < \infty$, then we have \[ \cos^2 \frac{\pi}{m_{i,j}} = \frac{a_{i,j} a_{j,i}}{4}. \]
  • The Weyl group is the Coxeter group associated with the Coxeter graph of $I$.
