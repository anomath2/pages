Cartan Data of Type $C$

Definition

Let
  • $n \in \mathbb{Z}_{\geq 2}$.
The Cartan datum of type $C_n$ is the Cartan datum $I = \{ 1,2,\dots,n \}$ such that \[ i \cdot j = \begin{cases} 2 & \text{ if } i = j, \\ -1 & \text{ if } |i-j| = 1 \text{ and } (i,j) \neq (n-1,n), \\ -2 & \text{ if } (i,j) = (n-1,n), \\ 0 & \text{ otherwise} \end{cases} \ \text{ for all } i,j \in I. \]
A Cartan datum is said to be of type $C$ if it is proportional to the Cartan datum of type $C_n$.
