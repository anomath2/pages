1.2. The Classical Lie Algebras

1.2.1. General and Special Linear Lie Algebras

If $V$ is a vector space over $\mathbb{F}$, we write $\mathfrak{gl}(V)$ for $\operatorname{End}(V)$ looked upon as a Lie algebra under the commutator. Similarly, we write $\mathfrak{gl}(n, \mathbb{F})$ for $M_n(\mathbb{F})$. These Lie algebras are called the general linear Lie algebras.
The special linear Lie algebras are \[ \mathfrak{sl}(V) := \{ T \in \mathfrak{gl}(V) \mid \operatorname{tr} T = 0 \}, \quad \mathfrak{sl}(n, \mathbb{F}) := \{ T \in \mathfrak{gl}(n, \mathbb{F}) \mid \operatorname{tr} T = 0 \}. \]

1.2.2. Lie Algebras Associated with Bilinear Forms

Let $V$ be a vector space over $\mathbb{F}$ and let $B$ a bilinear form on $V$. We define \[ \mathfrak{so}(V, B) := \{ X \in \mathfrak{gl}(V) \mid B(Xv, w) + B(v, Xw) = 0 \text{ for all } v,w \in V \}. \]

Orthogonal Lie Algebras

For $n \in \mathbb{Z}_{> 0}$, set \[ \mathfrak{so}(n, \mathbb{F}) := \{ X \in \mathfrak{gl}(n, \mathbb{F}) \mid {}^t X = -X \}. \]For $p,q \in \mathbb{Z}_{\geq 0}$ with $p+q = n$, set \[ \mathfrak{so}(p,q) := \{ X \in \mathfrak{gl}(n, \mathbb{R}) \mid {}^t X I_{p,q} + I_{p,q} X = O \}. \]
Let $V$ be an $n$-dimensional vector space, and $B$ a symmetric nondegenerate bilinear form on $V$. When $\mathbb{F} = \mathbb{C}$, then $\mathfrak{so}(V,B) \simeq \mathfrak{so}(n, \mathbb{C})$. When $\mathbb{F} = \mathbb{R}$ and the signature of $B$ is $(p,q)$, then $\mathfrak{so}(V,B) \simeq \mathfrak{so}(p,q)$.

Symplectic Lie Algebra

Set \[ \mathfrak{sp}(n, \mathbb{F}) := \{ X \in \mathfrak{gl}(2n, \mathbb{F}) \mid {}^t X J + JX = O \}. \]
Let $V$ be a $2n$-dimensional vector space, and $B$ a skew-symmetric nondegenerate bilinear form on $V$. Then, $\mathfrak{so}(V, B) \simeq \mathfrak{sp}(n, \mathbb{F})$.

Unitary Lie Algebras

For $p+q = n$, set \[ \mathfrak{u}(p,q) := \{ X \in \mathfrak{gl}(n, \mathbb{C}) \mid X^* I_{p,q} + I_{p,q}X = O \}. \]Note that this is a real subspace of $\mathfrak{gl}(n, \mathbb{C})$.
Set \[ \mathfrak{su}(p,q) := \mathfrak{sl}(n,\mathbb{C}) \cap \mathfrak{u}(p,q). \]
Let $V$ be an $n$-dimensional $\mathbb{C}$-vector space, and $B$ a nondegenerate Hermitian form on $V$. Set \[ \mathfrak{u}(V,B) := \{ T \in \mathfrak{gl}(V) \mid B(Tv, w) + B(v,Tw) = 0 \ \text{ for all } v,w \in V \}, \]and \[ \mathfrak{su}(V, B) := \mathfrak{sl}(V) \cap \mathfrak{u}(V, B). \]Then, $\mathfrak{u}(V,B) \simeq \mathfrak{u}(p,q)$ and $\mathfrak{su}(V,B) \simeq \mathfrak{su}(p,q)$, where $(p,q)$ denotes the signature of $B$.

1.2.4. Quaternionic Lie Algebras

Consider the $n \times n$ matrices over $\mathbb{H}$ with the usual matrix commutator. We will denote this Lie algebra by $\mathfrak{gl}(n, \mathbb{H})$.
Set \[ \mathfrak{su}^*(2n) := \{ X \in \mathfrak{gl}(n, \mathbb{H}) \mid \operatorname{tr} X = 0 \}. \]

Quaternionic Unitary Lie Algebras

For $p+q = n$, set \[ \mathfrak{sp}(p,q) := \{ X \in \mathfrak{gl}(n, \mathbb{H}) \mid X^* I_{p,q} + I_{p,q} X = O \}. \]

The Lie Algebra $\mathfrak{so}^*(2n)$

Set \[ \mathfrak{so}^*(2n) := \{ X \in \mathfrak{so}(2n, \mathbb{C}) \mid -J \overline{X} J = X \}. \]
