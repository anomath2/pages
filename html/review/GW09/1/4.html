1.4. Linear Algebraic Groups

Since each classical group $G \subset GL_n(\mathbb{F})$ is defined by algebraic equations, we can also study $G$ using algebraic techniques instead of analysis. We will take the field $\mathbb{F} = \mathbb{C}$ in this setting.

1.4.1 Definitions and Examples

Definition 1.4.1

A subgroup $G$ of $GL(n,\mathbb{C})$ is a linear algebraic group if there is a set $\mathcal{A}$ of polynomial functions on $M_n(\mathbb{C})$ such that \[ G = \{ g \in GL(n, \mathbb{C}) \mid f(g) = 0 \text{ for all } f \in \mathcal{A} \}. \]

1.4.2. Regular Functions

The set of regular functions on a linear algebraic group $G$ is denoted by $\mathcal{O}[G]$. For instance, \[ \mathcal{O}[GL(n,\mathbb{C})] = \mathbb{C}[x_{i,j}, \det(x)^{-1} \mid 1 \leq i,j \leq n]. \]
In general, \[ \mathcal{O}[G] \simeq \mathcal{O}[GL(n, \mathbb{C})]/ \mathcal{J}_G, \quad \mathcal{J}_G := \{ f \in \mathcal{O}[GL(n,\mathbb{C})] \mid f(G) = 0 \}. \]

1.4.3. Lie Algebra of an Algebraic Group

The next step in developing the theory of algebraic groups over $\mathbb{C}$ is to associate a Lie algebra to a linear algebraic group $G$. We want the definition to be purely algebraic ($\exp$ is analytic). Our strategy is to adapt the vector field point of view.

Definition 1.4.6

Let $\mathcal{A}$ be an algebra (not assumed to be associative) over a field $\mathbb{F}$. Then $\operatorname{Der}(\mathcal{A}) \subset \operatorname{End}(\mathcal{A})$ is the set of all linear transformations $D$ that satisfy \[ D(ab) = D(a)b + a D(b) \text{ for all } a,b \in \mathcal{A}. \]We call $D$ a derivation of $\mathcal{A}$

Let $G \subset GL(n,\mathbb{C})$ be a linear algebraic group. We define its Lie algebra $\mathfrak{g}$ by \[ \mathfrak{g} := \{ A \in \mathfrak{gl}(n, \mathbb{C}) \mid X_A f \in \mathcal{J}_G \text{ for all } f \in \mathcal{J}_G \}, \]where \[ X_A := \sum_{i,j=1}^n a_{i,j} \sum_{k=1}^n x_{k,i} \frac{\partial}{\partial x_{k,j}} \text{ if } A = (a_{i,j}). \]
