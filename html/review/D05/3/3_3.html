3.3. The Transform

For $f \in L^1_\mathrm{bc}(\mathbb{R})$, define its Fourier transform $\hat{f}$ by \[ \hat{f}(y) := \int_{-\infty}^\infty f(x) e^{-2\pi i xy} dx. \]

Theorem 3.3.1

Let $f \in L^1_\mathrm{bc}(\mathbb{R})$.
  • (a) If $g(x) = f(x) e^{2\pi i ax}$ for some $a \in \mathbb{R}$, then $\hat{g}(y) = \hat{f}(y-a)$.
  • (b) If $g(x) = f(x-a)$ for some $a \in \mathbb{R}$, then $\hat{g}(y) = \hat{f}(y) e^{-2\pi i ay}$.
  • (c) If $g \in L^1_\mathrm{bc}(\mathbb{R})$ and $h = f * g$, then $\hat{h}(y) = \hat{f}(y) \hat{g}(y)$.
  • (d) If $g(x) = f(x/\lambda)$ for some $\lambda > 0$, then $\hat{g}(y) = \lambda \hat{f}(y)(\lambda y)$.
  • (g) If $f$ is two times continuously differentiable and the functions $f', f''$ are all in $L^1_\mathrm{bc}(\mathbb{R})$, then $\hat{f} \in L^1_\mathrm{bc}(\mathbb{R})$.

Let $\mathcal{S} = \mathcal{S}(\mathbb{R})$ denote the space of Schwartz functions, i.e., the set of all infinitely differentiable functions $f: \mathbb{R} \rightarrow \mathbb{C}$ such that for every $m,n \geq 0$, we have \[ \sigma_{m,n}(f) := \sup_{x \in \mathbb{R}} |x^m f^{(n)}(x)| < \infty. \]
For example, \[ e^{-x^2} \in \mathcal{S}. \]

Proposition 3.3.3

We have $\mathcal{S} \subset L^1_\mathrm{bc}(\mathbb{R})$, and for each $f \in \mathcal{S}$, we have $\hat{f} \in \mathcal{S}$.
