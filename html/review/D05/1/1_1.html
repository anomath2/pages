[D05] 1.1. Periodic Functions

A function \[ f: \mathbb{R} \rightarrow \mathbb{C} \]is called periodic of period $L > 0$ if for each $x \in \mathbb{R}$, \[ f(x+L) = f(x). \]
If $f$ is periodic of period $L$, then the function \[ F: \mathbb{R} \rightarrow \mathbb{C};\ x \mapsto f(Lx) \]is a periodic function of period $1$. In this case, the original function $f$ can be recovered by the new function $F$ by \[ f(x) = F(\frac{x}{L}). \]Hence, we only need to consider periodic functions of period $1$. Such functions are simply said to be periodic.

Examples

The functions \[ f(x) = \sin 2\pi x, \ f(x) = \cos 2 \pi x, \ f(x) = e^{2 \pi i x} \]are periodic.
Each function on the half-open interval $[0,1)$ can be uniquely extended to a periodic function.

The set of periodic functions form a vector space. Let $C(\mathbb{R}/\mathbb{Z})$ denote the subspace of all continuous periodic functions.
For $f,g \in C(\mathbb{R}/\mathbb{Z})$, set \[ \langle f, g \rangle := \int_0^1 f(x) \overline{g(x)} dx, \]where the bar means complex conjugation.

Lemma 1.1.1

The $\langle , \rangle$ defines an inner product on the $\mathbb{C}$-vector space $C(\mathbb{R}/\mathbb{Z})$.
