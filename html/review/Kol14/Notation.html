Notation

  • $A = (a_{i,j})_{i,j \in I}$: an indecomposable symmetrizable generalized Cartan matrix.
  • $\mathbf{A} := \mathbb{K}[q]_{(q-1)}$ (10.1).
  • $\operatorname{Aut}(A)$: the group of automorphisms on $A$.
  • $\operatorname{Aut}(A, X)$: the group of automorphisms on $A$ leaving $X$ stable for a subset $X \subset I$.
  • $\operatorname{Aut}(A, X)^\tau := \{ \sigma \in \operatorname{Aut}(A, X) \mid \sigma \tau = \tau \sigma \}$.
  • $\operatorname{Aut}(\mathfrak{g})$: the group of Lie algebra automorphisms on $\mathfrak{g}$.
  • $\operatorname{Aut}(\mathfrak{g},\mathfrak{g}')$: the group of automorphisms on $\mathfrak{g}$ which restrict to the identity on $\mathfrak{g}'$.
  • $\mathfrak{b}^\pm := \mathfrak{n}^\pm \oplus \mathfrak{h}$: the Borel subalgebra of $\mathfrak{g}$.
  • $B_J := B_{j_1} \cdots B_{j_n}$ for $J = (j_1,\dots,j_n) \in I^n$.
  • $B_{\mathbf{c},\mathbf{s}}$: the algebra in Definition 5.1.
  • $\mathcal{C} := \{ \mathbf{c} \in (\mathbb{K}(q)^\times)^{I \setminus X} \mid c_i = c_{\tau(i)} \text{ if } \tau(i) \neq i \text{ and } (\alpha_i, \Theta(\alpha_i)) = 0 \}$.
  • $\overline{\mathcal{C}} := \{ \mathbf{c} \in (\overline{\mathbb{K}(q)}^\times)^{I \setminus X} \mid c_i = c_{\tau(i)} \text{ if } \tau(i) \neq i \text{ and } (\alpha_i, \Theta(\alpha_i)) = 0 \}$.
  • $D = \operatorname{diag}(\epsilon_i)_{i \in I}$: a symmetrizing matrix for $A$ with $\epsilon_i$ coprime.
  • $\overline{\mathcal{D}} := \{ \mathbf{d} \in (\overline{\mathbb{K}(q)}^\times)^{I \setminus X} \mid d_i = 1 \text{ if } \tau(i) = i \text{ or } (\alpha_i, \Theta(\alpha_i)) = 0 \text{ or } i \notin I^* \}$.
  • $F_{i,j}(x,y) := \sum_{n=0}^{1-a_{i,j}} (-1)^n {1-a_{i,j} \brack n}_{q_i} x^{1-a_{i,j}-n} y x^n$.
  • $F_J := F_{j_1} \cdots F_{j_n}$ for $J = (j_1,\dots,j_n) \in I^n$.
  • $G$: the Kac-Moody group associated to $\mathfrak{g}'$.
  • $\mathfrak{g}$: the Kac-Moody algebra of $A$ over $\mathbb{K}$.
  • $\mathfrak{g}_\beta := \{ x \in \mathfrak{g} \mid [h,x] = \beta(h)x \text{ for all } h \in \mathfrak{h} \}$: the root space for $\beta \in \mathfrak{h}^*$.
  • $\mathfrak{g}' := [\mathfrak{g},\mathfrak{g}]$: the derived algebra.
  • $\hat{\mathfrak{g}}$: the untwisted affine Lie algebra for finite dimensional $\mathfrak{g}$.
  • $\mathfrak{h} := \mathbb{K} \otimes_{\mathbb{Z}} P^\vee$.
  • $\mathfrak{h}' := \mathfrak{g}' \cap \mathfrak{h}$.
  • $\tilde{H} := \operatorname{Hom}_{\text{grp}}(Q, \mathbb{K}^\times)$.
  • $\tilde{H} := \operatorname{Hom}_{\text{grp}}(Q, \mathbb{K}(q)^\times)$.
  • $I$: finite set.
  • $I_{\mathrm{ns}} := \{ i \in I \setminus X \mid \tau(i) = i \text{ and } \alpha_i(h_j) = 0 \text{ for all } j \in X \}$.
  • $I^* \subset I \setminus X$: a subset containing exactly one element of each $\tau$-orbit in $I \setminus X$.
  • $\mathbb{K}$: an algebraically closed field of characteristic $0$.
  • $\overline{\mathbb{K}(q)}$: the algebraic closure.
  • $\mathfrak{k} := \{ x \in \mathfrak{g} \mid \theta(x) = x \}$.
  • $\mathfrak{k}' := \{ x \in \mathfrak{g}' \mid \theta(x) = x \}$.
  • $K_i := K_{\epsilon_i h_i}$ for $i \in I$.
  • $(K_i;0)_q := \frac{K_i-1}{q-1}$.
  • $K_\beta := \prod_{i \in I} K_i^{n_i}$ for $\beta = \sum_{i \in I} n_i \alpha_i \in Q$.
  • $L(\mathfrak{g}) := \mathfrak{g} \otimes \mathbb{K}[t,t^{-1}]$: the loop algebra for finite dimensional $\mathfrak{g}$.
  • $\lambda_{i,j} := (1-a_{i,j}) \alpha_i + \alpha_j$ (Remark 5.12).
  • $m_i := \exp(e_i) \exp(-f_i) \exp(e_i) \in G$ for $i \in I$.
  • $m_{i,j}$: the order of $r_ir_j \in W$ for $i \neq j \in I$.
  • $\mathcal{M}_X$: the subalgebra of $U_q(\mathfrak{g})$ generated by $E_j,F_j,K_j^{\pm 1}$, $j \in X$.
  • $\mathcal{M}_X^+ := U^+ \cap \mathcal{M}_X$.
  • $\mathfrak{n}^\pm$: the subalgebra of $\mathfrak{g}$ generated by $e_i/f_i$, $i \in I$.
  • $\omega \in \operatorname{Aut}(\mathfrak{g})$: the Chevalley involution; \[ \omega(e_i) = -f_i,\ \omega(f_i) = -e_i,\ \omega(h) = -h. \]
  • $\omega$: the Chevalley involution on $U_q(\mathfrak{g})$; \[ \omega(E_i) = -F_i,\ \omega(F_i) = -E_i,\ \omega(K_h) = K_{-h}. \]
  • $\operatorname{Out} := \begin{cases} \operatorname{Aut}(A) & \text{ if } \text{$A$ is of finite type}, \\ \operatorname{Aut}(A) \cup \omega \operatorname{Aut}(A) & \text{ otherwise}. \end{cases}$
  • $P := \{ h \in \mathfrak{h}^* \mid \lambda(P^\vee) \subset \mathbb{Z} \}$.
  • $P^\vee$: a free abelian group of rank $2|I|-\operatorname{rank}A$ with basis $\{ h_i \mid i \in I \} \sqcup \{ d_s \mid s = 1,\dots,\operatorname{corank}A \}$.
  • $\Phi := \{ \beta \in \mathfrak{h}^* \setminus \{0\} \mid \mathfrak{g}_\beta \neq 0 \}$.
  • $\Phi^+ := \Phi \cap Q^+$.
  • $\Pi := \{ \alpha_i \mid i \in I \} \subset \mathfrak{h}^*$: a linearly independent subset such that \[ \alpha_j(h_i) = a_{i,j}, \quad \alpha_j(d_s) \in \{ 0,1 \}. \]
  • $\Pi^\vee := \{ h_i \mid i \in I \}$.
  • $\psi$: the algebra automorphism on $U_q(\mathfrak{g})$ in 4.1.
  • $q$: an indeterminate.
  • $q_i := q^{\epsilon_i}$ for $i \in I$.
  • $Q := \mathbb{Z} \Pi$.
  • $Q^+ := \mathbb{Z}_{\geq 0} \Pi$.
  • $Q^\Theta := \{ \beta \in Q \mid \Theta(\beta) = \beta \}$.
  • $Q^\vee := \mathbb{Z} \Pi^\vee$.
  • $\mathcal{S} := \{ \mathbf{s} \in \mathbb{K}(q)^{I \setminus X} \mid s_i \neq 0 \text{ implies } i \in I_{\mathrm{ns}} \text{ and } a_{i,j} \in -2\mathbb{Z}_{\geq 0} \text{ for all } j \in I_\mathrm{ns} \setminus \{i\} \}$.
  • $\overline{\mathcal{S}} := \{ \mathbf{s} \in \overline{\mathbb{K}(q)}^{I \setminus X} \mid s_i \neq 0 \text{ implies } i \in I_{\mathrm{ns}} \text{ and } a_{i,j} \in -2\mathbb{Z}_{\geq 0} \text{ for all } j \in I_\mathrm{ns} \setminus \{i\} \}$.
  • $s(X,\tau) \in \tilde{H}$: the element in (2.7).
  • $T_i := T''_{i,1}$ for $i \in I$.
  • $T_X$: the algebra automorphism on $U_q(\mathfrak{g})$ in 4.2.
  • $\theta(X,\tau) \in \operatorname{Aut}(\mathfrak{g})$: the element in Theorem 2.5.
  • $\theta_q(X, \tau)$: the algebra automorphism on $U_q(\mathfrak{g}')$ in Definition 4.3.
  • $\Theta: \mathfrak{h}^* \rightarrow \mathfrak{h}^*$: the linear map in (2.10).
  • $U_q(\mathfrak{g})$: the quantized enveloping algebra of $\mathfrak{g}$.
  • $U_q(\mathfrak{g})_\beta := \{ u \in U_q(\mathfrak{g}) \mid K_i u K_i^{-1} = q^{(\alpha_i, \beta) u \text{ for all } i \in I }\}$ for $\beta \in Q$.
  • $\mathcal{U}'_\mathbf{A}$: the $\mathbf{A}$-subalgebra of $U_q(\mathfrak{g}')$ generated by $E_i,F_i,K_i^{\pm 1}, (K_i;0)_q$ (10.1).
  • $\mathcal{U}'_1 := \mathbb{K} \otimes_\mathbf{A} \mathcal{U}'_\mathbf{A}$ (10.1).
  • $U^\pm$: the subalgebra of $U_q(\mathfrak{g})$ generated by $E_i/F_i$, $i \in I$.
  • $U^0$: the subalgebra of $U_q(\mathfrak{g})$ generated by $K_h$, $h \in P^\vee$.
  • $U^{0 \prime}$: the subalgebra of $U_q(\mathfrak{g})$ generated by $K_i^{\pm 1}$, $i \in I$.
  • $U^{0 \prime}_\Theta$: the subalgebra of $U_q(\mathfrak{g})$ generated by $K_\beta$, $\beta \in Q^\Theta$.
  • $W = \langle r_i \mid i \in I \rangle$: the Weyl group.
  • $W_X$: the parabolic subgroup of $W$ generated by $\{ r_j \mid j \in J \}$ for $X \subset I$.
  • $w_X$: the longest element of $W_X$ for $X \subset I$ of finite type.
  • $\operatorname{wt}(J) := \sum_{i=1}^n \alpha_{j_i}$ for $J = (j_1,\dots,j_n) \in I^n$.
  • $(,)$: a nondegenerate symmetric bilinear form on $\mathfrak{h}$ and $\mathfrak{h}^*$ such that \[ (h_i,h) = \frac{\alpha_i(h)}{\epsilon_i}, \quad (d_m, d_n) = 0. \]
  • $\nu \leq \mu$: the partial order on $Q$ defined by $\mu-\nu \in Q^+$.
  • $|J| := n$ for $J = (j_1,\dots,j_n) \in I^n$.
  • $\overline{x}$: the image of $x \in \mathcal{U}'_\mathbf{A}$ in $\mathcal{U}'_1$.
