2. Involutive automorphisms of the second kind

  • In principle, involutive automorphisms of the second kind were classified in [KW92].

2.2. Automorphisms of the first and second kind

$\sigma \in \operatorname{Aut}(\mathfrak{g})$ is said to be of the first kind if \[ \sigma(\mathfrak{b}^+) = g \cdot \mathfrak{b}^+ \text{ for some } g \in G, \]equivalently, \[ \dim(\sigma(\mathfrak{b}^+) \cap \mathfrak{b}^-) < \infty. \]
$\sigma \in \operatorname{Aut}(\mathfrak{g})$ is said to be of the second kind if \[ \sigma(\mathfrak{b}^+) = g \cdot \mathfrak{b}^- \text{ for some } g \in G, \]equivalently, \[ \dim(\sigma(\mathfrak{b}^+) \cap \mathfrak{b}^+) < \infty. \]
If $\dim \mathfrak{g} < \infty$, then automorphisms are both of the first and of the second kind. Otherwise, each automorphism is either of the first or of the second kind, but never both.
For $x \in \tilde{H}$, define $\operatorname{Ad}(x) \in \operatorname{Aut}(\mathfrak{g})$ by \[ \operatorname{Ad}(x)|_\mathfrak{h} = 1, \quad \operatorname{Ad}(x)(v) = x(\alpha) v \text{ for all } v \in \mathfrak{g}_\alpha. \]
Each $\sigma \in \operatorname{Aut}(A)$ defines $\sigma \in \operatorname{Aut}(\mathfrak{g})$ such that \[ \sigma(e_i) = e_{\sigma(i)}, \quad \sigma(f_i) = f_{\sigma(i)}. \]([KW92, 4.19])

Proposition 2.1

\[ \operatorname{Aut}(\mathfrak{g}) = \operatorname{Out} \ltimes (\operatorname{Aut}(\mathfrak{g},\mathfrak{g}') \times (\tilde{H} \ltimes G)). \]

Definition 2.3

A pair $(X, \tau)$ consisting of a subset $X \subset I$ of finite type and an element $\tau \in \operatorname{Aut}(A, X)$ is called admissible if
  • (1) $\tau^2 = 1$.
  • (2) The action of $\tau$ on $X$ coincides with the action of $-w_X$.
  • (3) If $j \in I \setminus X$ and $\tau(j) = j$, then $\alpha_j(\rho_X^\vee) \in \mathbb{Z}$.

If $\dim \mathfrak{g} < \infty$, then the pair $(X = I, \tau = -w_X)$ is always admissible. The Satake diagrams in [A62, pp. 32--33] describe all other admissible pairs for finite dimensional simple $\mathfrak{g}$.

(2.7)

Fix a total order on $I$. Define $s(X,\tau) \in \tilde{H}$ by \[ s(X,\tau)(\alpha_j) = \begin{cases} 1 & \text{ if } j \in X \text{ or } \tau(j) = j, \\ \sqrt{-1}^{\alpha_j(2\rho_X^\vee)} & \text{ if } j \notin X \text{ and } \tau(j) > j, \\ (-\sqrt{-1})^{\alpha_j(2\rho_X^\vee)} & \text{ if } j \notin X \text{ and } \tau(j) > j. \end{cases} \]

Theorem 2.5

Let $(X, \tau)$ be an admissible pair. Then, \[ \theta(X, \tau) := s(X,\tau) \circ \tau \circ \omega \circ m_X \]is an involutive automorphism on $\mathfrak{g}$ of the second kind.

Theorem 2.7

The map \[ (X, \tau) \mapsto \theta(X, \tau) \]gives a bijection between the set of $\operatorname{Aut}(A)$-orbits of admissible pairs for $\mathfrak{g}$ and the set of $\operatorname{Aut}(\mathfrak{g})$-conjugacy classes of involutive automorphisms of the second kind.

(2.10)

For each $h \in \mathfrak{h}$, \[ \theta(X,\tau)(h) = -w_X \tau(h). \]It induces \[ \Theta: \mathfrak{h}^* \rightarrow \mathfrak{h}^*;\ \alpha \mapsto -w_X \tau(\alpha). \]

Lemma 2.8

The Lie algebra $\mathfrak{k}$ is generated by \[ \{ e_i, f_i \mid i \in X \} \sqcup \{ h \in \mathfrak{h} \mid \theta(h) = h \} \sqcup \{ f_i + \theta(f_i) \mid i \in I \setminus X \}. \]Similarly, the Lie algebra $\mathfrak{k}'$ is generated by \[ \{ e_i, f_i \mid i \in X \} \sqcup \{ h \in \mathfrak{h}' \mid \theta(h) = h \} \sqcup \{ f_i + \theta(f_i) \mid i \in I \setminus X \}. \]

Corollary 2.9

Let $\mathbf{s} = (s_i)_{i \in I \setminus X} \in \mathbb{K}^{I \setminus X}$. The universal enveloping algebra $U(\mathfrak{k})$ is generated by \[ \{ e_i, f_i \mid i \in X \} \sqcup \{ h \in \mathfrak{h} \mid \theta(h) = h \} \sqcup \{ f_i + \theta(f_i) + s_i \mid i \in I \setminus X \}. \]

Remark 2.10

A complete set of defining relations of $U(\mathfrak{k})$ or $U(\mathfrak{k}')$ in terms of the generators can be written down, but there are no publication describing them for general admissible pairs.
