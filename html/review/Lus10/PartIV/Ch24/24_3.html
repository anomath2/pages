24.3. The Canonical Basis of ${}^\omega \Lambda_\lambda \otimes \Lambda_{\lambda'}$

Assumption

In this section, the root datum is $Y$-regular.

24.3.1

Let $\lambda,\lambda' \in X^+$. Define a partial order on $\mathbf{B}^2$ as follows. We have $(b_1,b'_1) \leq (b_2,b'_2)$ if and only if \[ \operatorname{tr} |b_1| - \operatorname{tr} |b'_1| = \operatorname{tr} |b_2| - \operatorname{tr} |b'_2| \]and either \[ \operatorname{tr} |b_1| < \operatorname{tr} |b_2| \text{ and } \operatorname{tr} |b'_1| < \operatorname{tr} |b'_2| \]or \[ b_1 = b_2 \text{ and } b'_1 = b'_2. \]

24.3.2

Let \[ \Psi := \Theta \circ \overline{\cdot}^{\otimes 2} : {}^\omega \Lambda_\lambda \otimes \Lambda_{\lambda'} \rightarrow {}^\omega \Lambda_\lambda \otimes \Lambda_{\lambda'}. \]

Theorem 24.3.3

Let $\mathcal{L}$ denote the $\mathbb{Z}[v^{-1}]$-submodule of ${}^\omega \Lambda_\lambda \otimes \Lambda_{\lambda'}$ generated by \[ \{ b^+ \xi_{-\lambda} \otimes (b')^- \eta_{\lambda'} \mid (b,b') \in \mathbf{B}(\lambda) \times \mathbf{B}(\lambda') \}. \]
  • (a) For each $(b_1, b'_1) \in \mathbf{B}(\lambda) \times \mathbf{B}(\lambda')$, there exists a unique element $(b_1 \diamondsuit b'_1)_{\lambda,\lambda'} \in \mathcal{L}$ such that \[ \Psi((b_1 \diamondsuit b'_1)_{\lambda,\lambda'}) = (b_1 \diamondsuit b'_1)_{\lambda,\lambda'} \text{ and } (b_1 \diamondsuit b'_1)_{\lambda,\lambda'} - b_1^+ \xi_{-\lambda} \otimes (b'_1)^- \eta_{\lambda'} \in v^{-1} \mathcal{L}. \]
  • (b) $(b_1 \diamondsuit b'_1)_{\lambda,\lambda'}$ is equal to $b_1^+ \xi_{-\lambda} \otimes (b'_1)^- \eta_{\lambda'}$ plus a linear combination of elements $b_2^+ \xi_{-\lambda} \otimes (b'_2)^- \eta_{\lambda'}$ with \[ (b_2,b'_2) \in \mathbf{B}(\lambda) \times \mathbf{B}(\lambda') \and (b_2,b'_2) < (b_1,b'_1) \] with coefficients in $v^{-1} \mathbb{Z}[v^{-1}]$.
  • (c) $\{ (b_1 \diamondsuit b'_1)_{\lambda,\lambda'} \mid (b_1,b'_1) \in \mathbf{B}(\lambda) \times \mathbf{B}(\lambda') \}$ forms a basis of $\mathcal{L}$.
  • (d) The natural homomorphism $\mathcal{L} \cap \Psi(\mathcal{L}) \rightarrow \mathcal{L}/v^{-1} \mathcal{L}$ is an isomorphism.
