37.1. Definition of the Symmetries

Proposition 37.1.2

  • (a) For each $u \in \mathbf{U}$, there is a unique element $u'' \in \mathbf{U}$ such that for each $M \in \mathcal{C}'$ and $z \in M$, we have \[ T''_{i,-e}(u'' z) = u T''_{i,-e}(z). \]
  • (b) The map $u \mapsto u''$ is an automorphism of $\mathbf{U}$, denoted by $T'_{i,e}$.
  • (c) For each $u \in \mathbf{U}$, there is a unique element $u' \in \mathbf{U}$ such that for each $M \in \mathcal{C}'$ and $z \in M$, we have \[ T'_{i,-e}(u' z) = u T'_{i,-e}(z). \]
  • (d) The map $u \mapsto u'$ is an automorphism of $\mathbf{U}$, denoted by $T''_{i,-e}$. It is the inverse of $T'_{i,e}$.

37.1.3

  • $T'_{i,e}(E_i) = -K_i^{e} F_i$.
  • $T'_{i,e}(F_i) = -E_i K_i^{-e}$.
  • $T'_{i,e}(E_j) = \sum_{r+s=-a_{i,j}} (-1)^r v_i^{er} E_i^{(r)} E_j E_i^{(s)}$ if $j \neq i$.
  • $T'_{i,e}(F_j) = \sum_{r+s=-a_{i,j}} (-1)^r v_i^{-er} F_i^{(s)} F_j F_i^{(r)}$ if $j \neq i$.
  • $T'_{i,e}(K_h) = K_{s_i(h)}$.
  • $T''_{i,-e}(E_i) = -F_i K_i^{-e}$.
  • $T''_{i,-e}(F_i) = -K_i^{e} E_i$.
  • $T''_{i,-e}(E_j) = \sum_{r+s=-a_{i,j}} (-1)^r v_i^{er} E_i^{(s)} E_j E_i^{(r)}$ if $j \neq i$.
  • $T''_{i,-e}(F_j) = \sum_{r+s=-a_{i,j}} (-1)^r v_i^{-er} F_i^{(r)} F_j F_i^{(s)}$ if $j \neq i$.
  • $T''_{i,-e}(K_h) = K_{s_i(h)}$.
