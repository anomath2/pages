3.2. Triangular Decomposition For $\mathbf{U}$

Corollary 3.2.5

Let $\mathbf{U}^0$ denote the unital associative $\mathbb{Q}(v)$-algebra with generators \[ \{ K_h \mid h \in Y \} \]subject to the following relations: For each $h,h' \in Y$, \[ K_0 = 1, \quad K_h K_{h'} = K_{h+h'}. \]
  • (a) The linear map \[ \mathbf{f} \otimes \mathbf{U}^0 \otimes \mathbf{f} \rightarrow \mathbf{U};\ x \otimes K_h \otimes y \mapsto x^- K_h y^+ \] is an isomorphism.
  • (b) The linear map \[ \mathbf{f} \otimes \mathbf{U}^0 \otimes \mathbf{f} \rightarrow \mathbf{U};\ x \otimes K_h \otimes y \mapsto x^+ K_h y^- \] is an isomorphism.

Corollary 3.2.6

The algebra homomorphism \[ (\cdot)^\pm : \mathbf{f} \rightarrow \mathbf{U}^\pm \]is an isomorphism.
The algebra homomorphism \[ \mathbf{U}^0 \rightarrow \mathbf{U};\ K_h \mapsto K_h \]is injective.
