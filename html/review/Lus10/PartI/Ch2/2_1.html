2.1. The Weyl Group

2.1.1

For $i \neq j \in I$, set \[ h(i,j) := \begin{cases} 2 & \text{ if } a_{i,j} a_{j,i} = 0, \\ 3 & \text{ if } a_{i,j} a_{j,i} = 1, \\ 4 & \text{ if } a_{i,j} a_{j,i} = 2, \\ 6 & \text{ if } a_{i,j} a_{j,i} = 3, \\ \infty & \text{ if } a_{i,j} a_{j,i} \geq 4. \end{cases} \]
If $h(i,j) < \infty$, then we have \[ \cos^2 \frac{\pi}{h(i,j)} = \frac{a_{i,j} a_{j,i}}{4}. \]
The braid group is the group $\mathrm{Br}$ with generators \[ \{ s_i \mid i \in I \} \]subject to the following relations for all $i \neq j \in I$ with $h(i,j) < \infty$: \[ \underbrace{s_i s_j s_i \cdots}_{\text{$h(i,j)$ factors}} = \underbrace{s_j s_i s_j \cdots}_{\text{$h(i,j)$ factors}} \]
The Weyl group is the quotient group $W$ of $\mathrm{Br}$ factored by the normal subgroup generated by \[ \{ s_i^2 \mid i \in I \}. \]It is a Coxeter group.

2.1.3

A Cartan datum $(I, \cdot)$ is said to be
  • symmetric if $i \cdot i = 2$ for all $i \in I$.
  • simply laced if $i \cdot j \in \{2,0,-1\}$ for all $i,j in I$.
  • irreducible if $I \neq \emptyset$ and for each $i \neq j \in I$, there exists a sequence $i = i_1, \dots, i_r = j$ in $I$ such that $i_k \cdot i_{k+1} \neq 0$ for all $k = 1,\dots,r-1$.
  • of finite type if the matrix $(a_{i,j})_{i,j \in I}$ is positive definite.
