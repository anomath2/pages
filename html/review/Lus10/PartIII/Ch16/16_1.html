16.1. Definition of the Operators $\tilde{\phi}_i$, $\tilde{\epsilon}_i$, $\tilde{F}_i$, $\tilde{E}_i$

Assumption

In this chapter, we fix $i \in I$.

16.1.1

Let $\mathcal{D}_i$ denote the category whose objects are $\mathbb{Q}(v)$-vector spaces $P$ provided with two linear maps $\epsilon_i, \phi_i: P \rightarrow P$ such that $\epsilon_i$ is locally nilpotent and \[ \epsilon_i \phi_i = v_i^2 \phi_i \epsilon_i + 1; \]the morphisms are linear maps commuting with $\epsilon_i, \phi_i$.
For each $N \in \mathbb{Z}_{\geq 0}$, set \[ P(N) := \phi_i^{(N)} P(0), \quad P(0) := \{ x \in P \mid \epsilon_i(x) = 0 \}. \]

Lemma 16.1.2

  • (c) We have $P = \bigoplus_{N \geq 0} P(N)$ as a vector space. Moreover, for each $N \in \mathbb{Z}_{\geq 0}$, \[ \phi_i^{(N)} : P(0) \rightarrow P(N) \] is a linear isomorphism.
  • (d) $\phi_i$ is injective.

16.1.3

Define linear maps $\tilde{\phi}_i, \tilde{\epsilon}_i: P \rightarrow P$ by \[ \tilde{\phi}_i(\phi_i^{(N)} y) := \phi_i^{(N+1)} y, \quad \tilde{\epsilon}_i(\phi_i^{(N)} y) := \phi_i^{(N-1)} y \text{ for all } y \in P(0). \]

Lemma 16.1.4

Let $M \in \mathcal{C}'_i$ and $x \in M^t$.
  • (a) We can write uniquely \[ x = \sum_{s \geq \min(0,-t)} F_i^{(s)} x_s, \] where $E_i x_s = 0$, $x_s \in M^{t+2s}$, and $x_s = 0$ for large enough $s$. We can write uniquely \[ x = \sum_{s \geq \min(0,t)} E_i^{(s)} x'_s, \] where $F_i x'_s = 0$, $x'_s \in M^{t-2s}$, and $x'_s = 0$ for large enough $s$.
  • (b) We have \[ \sum_{s \geq \min(0,-t)} F_i^{(s+1)} x_s = \sum_{s \geq \min(0,t)} E_i^{(s-1)} x'_s. \] We denote either of these sums by $\tilde{F}_i x$.
  • (c) We have \[ \sum_{s \geq \min(0,-t)} F_i^{(s-1)} x_s = \sum_{s \geq \min(0,t)} E_i^{(s+1)} x'_s. \] We denote either of these sums by $\tilde{E}_i x$.
The operators $\tilde{\phi}_i$, $\tilde{\epsilon}_i$, $\tilde{F}_i$, $\tilde{E}_i$ are called Kashiwara's operators.
