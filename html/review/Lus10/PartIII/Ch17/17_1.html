17.1. First Application to Tensor Product

Lemma 17.1.3

  • (a) If $(,)$ are admissible symmetric bilinear forms on $P$ and $M$, then the symmetric bilinear form on $P \otimes M$ given by \[ (x_1 \otimes x_2, y_1 \otimes y_2) := (x_1, y_1) (x_2, y_2) \] is admissible.
  • (b) If $(,)$ are admissible symmetric bilinear forms on $M$ and $M'$, then the symmetric bilinear form on $M \otimes M'$ given by \[ (x_1 \otimes x_2, y_1 \otimes y_2) := (x_1, y_1) (x_2, y_2) \] is admissible.

Corollary 17.1.15

Let $P \in \mathcal{D}_i$, $M \in \mathcal{C}'_i$, $x \in P$, $y \in M^n$ be such that \[ \epsilon_i x = 0, \quad E_i y = 0. \]For each $m \in \mathbb{Z}_{\geq 0}$, let $\mathcal{L}_m$ be the $\mathbb{Z}[v^{-1}]$-submodule of $P \otimes M$ generated by the vectors $\phi_i^{(s)} x \otimes F_i^{(s')} y$ with $s + s' = m$. Then, we have \begin{align*} \begin{split} &\tilde{\phi}_i(\phi_i^{(s)}x \otimes F_i^{(s')}y) = \phi_i^{(s)} x \otimes F_i^{(s'+1)} y \pmod{v^{-1} \mathcal{L}_{s+s'+1}} \text{ if } s+s' < n, \\ &\tilde{\phi}_i(\phi_i^{(s)}x \otimes F_i^{(s')}y) = \phi_i^{(s+1)} x \otimes F_i^{(s')} y \pmod{v^{-1} \mathcal{L}_{s+s'+1}} \text{ if } s+s' \geq n, \\ &\tilde{\epsilon}_i(\phi_i^{(s)}x \otimes F_i^{(s')}y) = \phi_i^{(s)} x \otimes F_i^{(s'-1)} y \pmod{v^{-1} \mathcal{L}_{s+s'-1}} \text{ if } s+s' \leq n, \\ &\tilde{\epsilon}_i(\phi_i^{(s)}x \otimes F_i^{(s')}y) = \phi_i^{(s-1)} x \otimes F_i^{(s')} y \pmod{v^{-1} \mathcal{L}_{s+s'-1}} \text{ if } s+s' > n \end{split} \end{align*}
