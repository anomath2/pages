17.2. Second Application to Tensor Products

Corollary 17.2.4

Let $M,N \in \mathcal{C}'_i$, $x \in M^p$, $y \in N^n$ be such that \[ E_i x = 0, \quad E_i y = 0. \]For each $m \in \mathbb{Z}_{\geq 0}$, let $\mathcal{L}_m$ be the $\mathbf{A}(\mathbb{Z})$-submodule of $M \otimes N$ generated by the vectors $F_i^{(s)} x \otimes F_i^{(s')} y$ with $s + s' = m$. Then, we have \begin{align*} \begin{split} &\tilde{F}_i(F_i^{(s)}x \otimes F_i^{(s')}y) = F_i^{(s)} x \otimes F_i^{(s'+1)} y \pmod{v^{-1} \mathcal{L}_{s+s'+1}} \text{ if } s+s' < n, \\ &\tilde{F}_i(F_i^{(s)}x \otimes F_i^{(s')}y) = F_i^{(s+1)} x \otimes F_i^{(s')} y \pmod{v^{-1} \mathcal{L}_{s+s'+1}} \text{ if } s+s' \geq n, \\ &\tilde{E}_i(F_i^{(s)}x \otimes F_i^{(s')}y) = F_i^{(s)} x \otimes F_i^{(s'-1)} y \pmod{v^{-1} \mathcal{L}_{s+s'-1}} \text{ if } s+s' \leq n, \\ &\tilde{E}_i(F_i^{(s)}x \otimes F_i^{(s')}y) = F_i^{(s-1)} x \otimes F_i^{(s')} y \pmod{v^{-1} \mathcal{L}_{s+s'-1}} \text{ if } s+s' > n \end{split} \end{align*}
