2.1. Examples and formulation of the problem

1.1. Main definitions and some examples

Let $f$ be an integrable function on an interval $[a,b]$ of length $L := b-a$. The $n$-th Fourier coefficient of $f$ is defined by \[ \hat{f}(n) := \frac{1}{L} \int_a^b f(x) e^{-2\pi i n x/L} dx, \quad n \in \mathbb{Z}. \]The Fourier series of $f$ is given formally by \[ \sum_{n = -\infty}^\infty \hat{f}(n) e^{2\pi i n x/L}. \]We write \[ f(x) \sim \sum_{n=-\infty}^\infty a_n e^{2\pi i n x/L} \]to indicate that $\hat{f}(n) = a_n$.
The $N$-th partial sum of the Fourier series of $f$ is \[ S_N(f)(x) := \sum_{n=-N}^N \hat{f}(n) e^{2\pi i n x/L}. \]

Example 1.

Let $f(\theta) = \theta$ for $-\pi \leq \theta \leq \pi$. Then, \[ f(\theta) \sim \sum_{n \neq 0} \frac{(-1)^{n+1}}{i n} e^{in \theta} = 2 \sum_{n=1}^\infty (-1)^{n+1} \frac{\sin n\theta}{n}. \]

Example 4.

The trigonometric polynomial \[ D_N(x) := \sum_{n = -N}^N e^{inx} = \frac{\sin (N+1/2)x}{\sin(x/2)}, \quad x \in [-\pi, \pi] \]is called the $N$-th Dirichlet kernel.

Example 5.

The Poisson kernel is \[ P_r(\theta) := \sum_{n=-\infty}^\infty r^{|n|} e^{in \theta} = \frac{1-r^2}{1-2r\cos \theta + r^2} \]for $0 \leq r < 1$, $\theta \in [-\pi, \pi]$.
