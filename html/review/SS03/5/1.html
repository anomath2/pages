5.1. Elementary theory of the Fourier transform

A function $f$ on $\mathbb{R}$ is said to be of moderate decrease if $f$ is continuous and there exists a constant $A > 0$ such that \[ |f(x)| \leq \frac{A}{1 + x^2} \text{ for all } x \in \mathbb{R}. \]Let $\mathcal{M}(\mathbb{R})$ denote the set of functions of moderate decrease on $\mathbb{R}$.

If $f \in \mathcal{M}(\mathbb{R})$, then we define its Fourier transform for $\xi \in \mathbb{R}$ by \[ \hat{f}(\xi) := \int_{-\infty}^\infty f(x) e^{-2\pi i x \xi} dx. \]We use the notation \[ f(x) \rightarrow \hat{f}(\xi) \]to mean that $\hat{f}$ denotes the Fourier transform of $f$.

The Schwartz space $\mathcal{S}(\mathbb{R})$ on $\mathbb{R}$ consists of the set of all indefinitely differentiable functions $f$ so that $f$ an all its derivatives $f', f'', \dots$, are rapidly decreasing in the sense that \[ \sup_{x \in \mathbb{R}} |x|^k |f^{(l)}(x)| < \infty \text{ for all } k,l \geq 0. \]

Proposition 1.2

Let $f \in \mathcal{S}(\mathbb{R})$.
  • (i) $f(x+h) \rightarrow \hat{f}(\xi) e^{2\pi i h \xi}$ for all $h \in \mathbb{R}$.
  • (ii) $f(x)e^{-2\pi i xh} \rightarrow \hat{f}(\xi+h)$ for all $h \in \mathbb{R}$.
  • (iii) $f(\delta x) \rightarrow \delta^{-1} \hat{f}(\delta^{-1} \xi)$ for all $\delta > 0$.
  • (iv) $f'(x) \rightarrow 2\pi i \xi \hat{f}(\xi)$.
  • (v) $-2\pi i x f(x) \rightarrow \frac{d}{d\xi}\hat{f}(\xi)$.

Theorem 1.3

If $f \in \mathcal{S}(\mathbb{R})$, then $\hat{f} \in \mathcal{S}(\mathbb{R})$.

Theorem 1.9 (Fourier inversion)

If $f \in \mathcal{S}(\mathbb{R})$, then \[ f(x) = \int_{-\infty}^\infty \hat{f}(\xi) e^{2\pi i x \xi} d\xi. \]

Proposition 1.11

If $f,g \in \mathcal{S}(\mathbb{R})$, then
  • (i) $f*g \in \mathcal{S}(\mathbb{R})$.
  • (ii) $f*g = g*f$.
  • (iii) $\widehat{f*g}(\xi) = \hat{f}(\xi) \hat{g}(\xi)$.

Theorem 1.12 (Plancherel)

If $f \in \mathcal{S}(\mathbb{R})$, then $||\hat{f}|| = ||f||$.
