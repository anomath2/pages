Definitions Concerning Generalized Cartan Matrices

Definition of Generalized Cartan Matrices

Let
  • $I$: a finite set.
An integer valued matrix $A = (a_{i,j})_{i,j \in I}$ is said to be a generalized Cartan matrix (GCM in short) if the following are satisfied for all $i,j \in I$.
  • $a_{i,i} = 2$.
  • $a_{i,j} \in \mathbb{Z}_{\leq 0}$ if $i \neq j$.
  • $a_{i,j} = 0$ implies $a_{j,i} = 0$.
([Kac90, $\S$1.1])

Definition of Morphisms between GCMs

Let
  • $A = (a_{i,j})_{i,j \in I}, B = (b_{i,j})_{i,j \in J}$: two generalized Cartan matrices.
A morphism from $A$ to $B$ is a map $f : I \rightarrow J$ such that \[ b_{f(i), f(j)} = a_{i,j} \text{ for all } i,j \in I. \]
A morphism is said to be an equivalence if the underlying map is bijective.
The generalized Cartan matrix $A$ is said to be a submatrix of $B$ if $I \subset J$ and the inclusion map $I \hookrightarrow J$ is a morphism of generalized Cartan matrices.

Definition of Decomposable GCMs

A GCM $A$ is said to be decomposable if there exists a nontrivial partition $I = I_1 \sqcup I_2$ such that \[ a_{i,j} = 0 \text{ for all } (i,j) \in I_1 \times I_2. \]

Definition of Indecomposable GCMs

A GCM is said to be indecomposable if it is not decomposable.

Definition of Symmetric GCMs

A GCM $A = (a_{i,j})_{i,j \in I}$ is said to be symmetric if it is a symmetric matrix, i.e., \[ a_{i,j} = a_{j,i} \ \text{ for all } i,j \in I. \]

Definition of Symmetrizable GCMs

A GCM $A = (a_{i,j})_{i,j \in I}$ is said to be symmetrizable if there exists a diagonal matrix $D = \operatorname{diag}(d_i \mid i \in I) \in \mathrm{Mat}_I(\mathbb{Z})$, called a symmetrizing matrix, such that
  • $d_i > 0$ for all $i \in I$.
  • $DA$ is a symmetric matrix, i.e., \[ d_i a_{i,j} = d_j a_{j,i} \ \text{ for all } i,j \in I. \]
([Kac90, Section 2.1])

Memo

  • A generalized Cartan matrix is a generalization of the Cartan matrix of a complex semisimple Lie algebra. In fact, the Cartan matrices are exactly the generalized Cartan matrices of finite types.
  • A generalized Cartan matrix is used to construct a Kac-Moody algebra, a quantum group.
